Italico

Marcação de texto

a marcação de texto é usada com um marca texto

Palavras grandes e pequenas

para as letras grandes podemos usar a tag <Big> (está obsoleta)

para as letras pequenas podemos usar a tag <Small>

Deletado e inserido

para textos deletado (exemplo) podemos usar a tag <del>

e para textos inserido (exemplo) podemos usar a tag <ins>
